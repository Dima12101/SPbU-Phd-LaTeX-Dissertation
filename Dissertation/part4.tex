\chapter{ГИБРИДНЫЙ МЕТОД (MA-VCG-QMIX)}\label{sec:ch4}

\section{Обоснование гибридного подхода}

Анализ существующих методов показал, что ни один из подходов не обеспечивает одновременно адаптивность, справедливость и устойчивость к неполной информации.

Аукционные механизмы (в частности, VCG) гарантируют стратегическую правдивость и максимизацию социального благосостояния, однако не учитывают динамику среды. Методы многоагентного обучения с подкреплением (MARL), напротив, обеспечивают адаптивность, но не гарантируют справедливость распределения ресурсов и могут приводить к стратегически неоптимальному поведению агентов.

В связи с этим предлагается гибридный метод, объединяющий:
\begin{itemize}
	\item аукционный механизм распределения ресурсов (VCG);
	\item модель Dec-POMDP;
	\item алгоритм многоагентного обучения с подкреплением (QMIX).
\end{itemize}

\section{Концепция гибридного метода}

Предлагаемый метод основан на разделении процесса управления на два уровня:

\begin{itemize}
	\item \textbf{экономический уровень} — формирование справедливого распределения ресурсов посредством аукционного механизма;
	\item \textbf{адаптивный уровень} — корректировка стратегий агентов на основе обучения с подкреплением.
\end{itemize}

Таким образом, система рассматривается как иерархическая многоагентная структура:

\begin{equation}
	\mathcal{H} = \langle \mathcal{A}_{auction}, \mathcal{A}_{RL}, \mathcal{E} \rangle,
\end{equation}

где:
\begin{itemize}
	\item $\mathcal{A}_{auction}$ — аукционный слой;
	\item $\mathcal{A}_{RL}$ — обучающий слой;
	\item $\mathcal{E}$ — среда.
\end{itemize}

В предложенном гибридном методе аукционный слой используется для формирования распределения ресурсов и экономического сигнала, а MARL-компонента обучает агентные стратегии принятия решений в условиях частичной наблюдаемости и динамической среды. Таким образом, объектом обучения является не сам VCG-механизм, а политика агентного поведения в повторяющемся процессе распределения ресурсов при наличии аукционного сигнала.

\section{Формализация гибридной модели}

\subsection{Расширенная Dec-POMDP модель}

Модель расширяется за счет включения экономического механизма:

\begin{equation}
	\text{Dec-POMDP}' = \langle N, S, A, P, R', O, \Omega, \gamma, h \rangle,
\end{equation}

где новая функция вознаграждения определяется как:

\begin{equation}
	R' = R_{RL} + \lambda \cdot R_{auction}.
\end{equation}

Здесь:
\begin{itemize}
	\item $R_{RL}$ — стандартное вознаграждение MARL;
    \item $R_{auction}$ — вклад аукционного механизма;
	\item $\lambda$ — коэффициент балансировки.
\end{itemize}

\subsection{Интеграция аукционного механизма}

Аукционный механизм формирует распределение ресурсов:

\begin{equation}
	x^* = \arg\max_x SW(x),
\end{equation}

где $SW(x)$ определяется в главе 2.

Результат аукциона влияет на:
\begin{itemize}
	\item допустимое пространство действий агентов;
	\item функцию вознаграждения;
	\item состояние среды.
\end{itemize}

Алгоритм функционирует в двух режимах: \textbf{режиме онлайн-исполнения} (Online Phase) и \textbf{режиме оффлайн-обучения} (Offline Phase).

\subsection{Режим 1: Онлайн-исполнение}

Онлайн-фаза работает в реальном времени, без доступа к глобальному состоянию системы. 
\begin{enumerate}
	\item \textbf{Формирование задач и наблюдений}. В момент времени $t$, каждое устройство $d_i$ формирует список задач $\tau$, а каждый периферийный узел $n_j$ при этом формирует своё локальное наблюдение $o_j^t$.
	\item \textbf{Проведение аукциона}:
	\begin{itemize}
		\item сбор заявок $b_i(t)$ на выполнение задач;
		\item рассчет стомости $C_j(t)$ и полезности $U_i^t(\tau)$ выполнение задач;
		\item определение оптимального распределения $x^*$.
	\end{itemize}
	\item \textbf{Формирование вознаграждения с учетом VCG}: Для каждого устройства $d_i$  расчитываются VCG платежи $p_i$. Рассчитанные платежи интегрируются в функцию вознаграждения $r_j$ узлов. 
	\item \textbf{Назначение задач на узлы}. Каждый периферийный узел принимает решение о взятии задачи на исполнение на основе своего локального наблюдения:
	\begin{enumerate}
		\item \textbf{Вычисление локальной Q-функции}. Каждый узел имеет свою нейросеть, которая вычисляет Q-значения $\mathbf{Q}_j(t)$ для всех возможных действий $a_j$.
		\item \textbf{Выбор действия и исполнение}. Узел выбирает действие на основе $\epsilon$-жадной стратегии, далее либо узел берет задачу на исполнение, либо нет.
	\end{enumerate} 
	\item \textbf{Накопление опыта}. Каждый переход сохраняется в Experience Reply Buffer: $o^t, a^t, r^t, o^{t+1}$.
\end{enumerate}

\subsection{Режим 2: Оффлайн-обучение}

Оффлай-фаза включает \textbf{обучение сетей QMIX}. На основе Experience Reply Buffer с обновлёнными вознаграждениями, система обучает сети QMIX путём минимизации TD-ошибки.

\subsection{Псевдокод алгоритма}

\begin{verbatim}
	Initialize Q-networks and replay buffer
	for each episode do
	observe state s
	for each agent i do
	select action a_i using epsilon-greedy
	end for
	
	run auction mechanism:
	create bids b_i(t)
	compute costs C_j(t) and utilities U_i(task)
	determine allocation x*
	
	execute actions and observe reward r
	store transition (s, a, r, s') in buffer
	
	sample minibatch from buffer
	update Q-networks using QMIX
	
	update target networks
	end for
\end{verbatim}

\subsection{Поток данных: пример со сценарием}

\textbf{Время $t=0$ (онлайн-фаза)}.

Устройство $d_1$ приходит с задачей $\tau_1$ (CPU=2, важность=0.8). Узлы имеют нагрузку 30\% и 20\% соответственно.

После проведения аукциона задача распределяется на узел. После вычисления Q-функций узлы выбирают действия. Задача $\tau_1$ назначается узлу с максимальным Q-значением для действия Accept. Система получает вознаграждение и сохраняет переход в буфер.

\textbf{Оффлайн-фаза (после 1000 переходов)}.

Система анализирует буфер и обнаруживает закономерности:
\begin{itemize}
	\item когда узел перегружен (load $> 0.4$), отклонение задач приводит к лучшему результату;
	\item платежи варьируются от 0.5 до 0.8 в зависимости от важности.
\end{itemize}

После чего система переобучает сети QMIX.

\section{Свойства предложенного метода}

Предлагаемый гибридный метод MA-VCG-QMIX обладает рядом свойств, которые,
однако, имеют различную природу и требуют различной степени обоснования.

Во-первых, в рамках одного статического раунда распределения ресурсов
аукционный слой метода наследует свойства классического механизма
Викри--Кларка--Гровса. При фиксированных истинных функциях полезности и
стоимости, а также при точном решении задачи максимизации социального
благосостояния, выполняется \textbf{доминантно-стратегическая совместимость
аукционного слоя}. При этом данное свойство относится только к отдельному
раунду распределения и не переносится автоматически на весь повторяющийся
процесс управления с обучением.

Во-вторых, \textbf{адаптивность} метода обеспечивается использованием кооперативного
многоагентного обучения с подкреплением. Агенты формируют стратегии выбора
действий на основе локальных наблюдений и накопленного опыта взаимодействия
со средой, что позволяет учитывать изменение нагрузки, вариативность сетевых
задержек и отказоустойчивость узлов. Данное свойство носит не дедуктивный, а
конструктивный характер и подтверждается результатами экспериментального
исследования.

В-третьих, \textbf{возможность работы в условиях частичной наблюдаемости} определяется
самой формализацией задачи в виде расширенного Dec-POMDP и реализацией
парадигмы централизованного обучения и децентрализованного исполнения.
На этапе обучения используется расширенная информация о состоянии системы, а
на этапе исполнения каждый агент принимает решение только на основе локально
доступных наблюдений.

В-четвёртых, \textbf{справедливость распределения, устойчивость к неполной информации
и масштабируемость} рассматриваются в данной работе как свойства, подлежащие
эмпирической оценке. Для их анализа используются значения индекса Джайна,
характеристики времени отклика, доля задач, выполненных в срок, а также
изменение качества работы системы при усложнении сценариев и росте числа
агентов.

Наконец, \textbf{строгая глобальная оптимальность} гибридного метода в общем случае не
утверждается. Если в аукционном слое оптимальность связана с максимизацией
социального благосостояния, то в полном гибридном контуре она ограничивается
аппроксимационным характером обучения с подкреплением и стохастичностью
многоагентной среды.
\begin{equation}
	\text{LocalDecision}(a_j) \approx \text{GlobalOptimal}(x^*)
\end{equation}

Следовательно, основное преимущество предложенного метода состоит не в
наличии универсальных строгих гарантий для всех его компонентов, а в
объединении экономически интерпретируемого механизма распределения ресурсов
с адаптивным обучаемым контуром, ориентированным на работу в динамической
частично наблюдаемой edge-среде.

\section{Анализ вычислительной сложности}

Сложность метода определяется как:

\begin{equation}
	O(k \cdot |A| + Auction(k)),
\end{equation}

где:
\begin{itemize}
	\item $|A|$ — размер пространства действий;
	\item $Auction(k)$ — сложность аукционного механизма.
\end{itemize}

В случае использования эффективной реализации аукциона:

\begin{equation}
	Auction(k) = O(k \log k).
\end{equation}

Таким образом, общий алгоритм сохраняет линейно-логарифмическую масштабируемость.

\section{Отличие от существующих подходов}

Предложенный метод отличается от известных подходов тем, что

\begin{itemize}
	\item рассматривает распределение ресурсов как повторяющийся многоагентный процесс с частичной наблюдаемостью;
	\item использует аукционный механизм VCG как источник экономического сигнала в контуре обучения;
	\item реализует обучение кооперативных агентных стратегий в парадигме CTDE;
	\item ориентирован на сценарии динамической нагрузки и ограниченной локальной информации.
\end{itemize}

В данной главе разработан гибридный метод адаптивного управления ресурсами, объединяющий аукционные механизмы и многоагентное обучение с подкреплением.

Предложенный метод:
\begin{itemize}
	\item формализован в виде расширенной Dec-POMDP модели;
	\item реализует интеграцию VCG и QMIX;

%% 9.2.3 В заключении автореферата диссертации излагают итоги данного исследования, рекомендации и перспективы дальнейшей разработки темы.
\begin{enumerate}
	\item Проведен анализ существующих методов управления ресурсами в распределенных системах, выявлены их ограничения, связанные с отсутствием адаптивности, недостаточной масштабируемостью и невозможностью обеспечения справедливого распределения ресурсов.
	
	\item Разработана математическая модель распределения вычислительных ресурсов, учитывающая:
	\begin{itemize}
		\item гетерогенность вычислительных узлов;
		\item ограничения по задержкам;
		\item динамику нагрузки;
		\item экономические аспекты взаимодействия агентов.
	\end{itemize}
	
	\item Формализована задача управления ресурсами в виде децентрализованного частично наблюдаемого марковского процесса принятия решений (Dec-POMDP), позволяющего учитывать неполную информацию и совместное поведение агентов.
	
	\item Разработан гибридный метод адаптивного управления ресурсами, объединяющий:
	\begin{itemize}
		\item аукционный механизм Викри–Кларка–Гровса;
		\item методы многоагентного обучения с подкреплением;
		\item архитектуру QMIX.
	\end{itemize}
	
	\item Предложен алгоритм MA-VCG-QMIX, обеспечивающий интеграцию экономических стимулов и адаптивного поведения агентов.
	
	\item Проведена экспериментальная оценка, показавшая, что предложенный метод:
	\begin{itemize}
		\item снижает среднее время отклика;
		\item увеличивает долю задач, выполненных в срок;
		\item повышает справедливость распределения ресурсов;
		\item демонстрирует устойчивость к отказам и динамической нагрузке.
	\end{itemize}
	
	\item Показана практическая применимость метода в роботизированных складских системах, интеллектуальных транспортных системах и промышленных IoT-средах.
\end{enumerate}
